Basic Raymarcher

Not long ago, I found Shadertoy. Since then I've been fascinated with what you can do with just a fragment shader. Using just a fragment shader means that you have to be clever about how to specify the 3D geometry of your scene because the only global state (besides the uniforms) is your code. So, how is this done?

Model Specification

Using standard OpenGL, models are specified with vertices. This won't work here because vertices require state. Instead, we have to specify everything implicitly through a function. This function should take a position vec3 and return a float, specifying the distance from the position to the model. The distance at the model's boundary is 0, and the distance is negative when a position is inside the model. This is a great way to specify a model in a global way, but it isn't really obvious how you might display it.
